# !!SennyinQoL

## Overview
SennyinQoL is a lightweight World of Warcraft UI addon that provides user interface enhancements.


## Feats
This addon is designed to improve the WoW interface by giving you:
- Automatic UI scaling.
- Class/reaction colours for default Player, Target, Focus, Target of Target frames.
- Enable shorthand commands for:
    - /rl to reload
    - /lg to leave group
    - /rc to ready check
- Centering of Blizzard Cooldown Manager frames (Essential, Utility, Buff Icon) and ability to anchor them together
- Nameplate colouring (work in progress)
- Tooltip improvements (not yet working)
- Personal Resource Display improvements - anchor to essential cooldowns and match width
- Custom Power, Class Resource bars
- Class Buff Bars for tracking those with stacks - currently only working for Fury Warrior Whirlwind.

## Notes
Place this folder in your World of Warcraft `_retail_` Interface/AddOns directory and enable it from the in-game AddOns menu.
